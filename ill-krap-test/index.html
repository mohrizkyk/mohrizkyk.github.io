ILL KraP Test

Try The App

About

It Looks Like Kraepelin-Pauli Test (ILL KraP Test) is a simple test of adding 2 numbers and take only the last digit as the answer (the modulo of 10).

After completing the test, user is able to view the summary of their test result which is based on their calculation time.

User is also able to download the test records for further analysis on their own, since maybe the theory used in this app to interpret the data is incorrect.
